Block Level ელემენტები: ეს ელემენტები ხაზზე სრულ სივრცეს იკავებს Block ელემენტები: ეს არის HTML ელემენტები, რომლებიც: ყოველთვის იწყებენ ახალ ხაზს. იკავებენ მთელ სიგანეს (მშობელი კონტეინერის ჩარჩოში). გამოიყენება გვერდის ძირითადი სტრუქტურის შესაქმნელად. Inline ელემენტები: ეს ელემენტები იმავე ხაზზე რჩებიან და სხვა ელემენტებთან ერთად განლაგდებიან ეს არის ელემენტები, რომლებიც: განლაგებულია ერთ ხაზზე. იკავებენ მხოლოდ იმდენ ადგილს, რამდენიც მათ შიგთავსს სჭირდება. გამოიყენება ტექსტის ან შინაარსის მცირე ნაწილების გამოსაყოფად ან სტილიზაციისთვის. Box Model არის სტრუქტურული მოდელი, რომელიც განსაზღვრავს, როგორ უნდა განლაგდეს თითოეული ელემენტი გვერდზე. ყუთი 4 ნაწილად იყოფა: (Content): ტექსტი ან სურათი, რაც ელემენტშია. (Padding): სივრცე შიგთავსსა და საზღვარს შორის. (Border): ელემენტის გარშემო ხაზი. (Margin): სივრცე ელემენტსა და სხვა ელემენტებს შორის. Box Model გვეხმარება, სწორად მოვათავსოთ და მოვაწესრიგოთ ელემენტები.
